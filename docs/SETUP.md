# Setup local

## Requisitos
- Unity Hub
- Unity 2022 LTS (ou superior)
- Git

## Passos
1. Criar/abrir projeto Unity com raiz em `src-unity`
2. Garantir que a pasta `Assets/Scripts` esteja presente
3. Copiar os scripts desta base para o projeto
4. Criar cena `Main.unity` com:
   - Tabuleiro
   - Camera orbit
   - GameController em um objeto vazio
5. Configurar build target Android (depois iOS)
